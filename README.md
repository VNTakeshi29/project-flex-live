# Project Flex

Personal site built with Next.js (App Router), TypeScript, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy

- Deploy on Vercel or any Node hosting that supports Next.js.
- Required assets:
  - `public/music.mp3` (optional `public/music.ogg`)
  - `public/rain.mp4`
